import Tabs from "@theme/Tabs";
import TabItem from "@theme/TabItem";

First, you need to install the `@airgap/beacon-sdk` package.

<Tabs
  defaultValue="npm"
  values={[
    { label: "NPM", value: "npm" },
    { label: "Yarn", value: "yarn" },
  ]}
>
<TabItem value="npm">

```bash
npm install --save @airgap/beacon-sdk
```

</TabItem>
<TabItem value="yarn">

```bash
yarn add @airgap/beacon-sdk
```

</TabItem>
</Tabs>

If you plan to interact with contracts, you will most likely also want to install `@taquito`.

<Tabs
  defaultValue="npm"
  values={[
    { label: "NPM", value: "npm" },
    { label: "Yarn", value: "yarn" },
  ]}
>
<TabItem value="npm">

```bash
npm install --save @airgap/beacon-sdk @taquito/taquito @taquito/beacon-wallet
```

</TabItem>
<TabItem value="yarn">

```bash
yarn add @airgap/beacon-sdk @taquito/taquito @taquito/beacon-wallet
```

</TabItem>
</Tabs>
